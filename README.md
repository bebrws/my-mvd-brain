# Open Source Apache 2.0 memvid CLI and Harness Context/Skills for 3 Harnesses
I have been thinking about building something very similar to this project for a while.
And then found this project memvid that looks to have everything I want.
Except.
There really is vendor lockin. the CLI is not opensource. There are dependencies on a 3rd party API.
The goal here is to create an Apache 2.0 licenced version of memvid that can be used by any AI agent.

## CLI
In this repo I have created a CLI. You can build the CLI, named `mvd` as to not interfere if you want to use the `memvid` CLI. 
There are also repositories for how this could be used with 3 promeinent harnesses.
Antigravity, Cursor, and Claude Code.

## Why?
memvid is amazing. The creators did amazing work. But I do not want to spend time working with a tool that is partially open source.
I am sure there will be issues with the tool. The cli was generated by a prompt asking to add the same features that the closed source CLI has. As this was done by reading the documentation and not reverse engineering the original CLI I believe this is clean room coded.
Hoepefully this will work as intended and can be fixed if not.
